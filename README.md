# Issue Tracker

This repository is organized for two parallel workstreams:

- `Backend/` contains the Spring Boot API and persistence layer.
- `frontend/` is reserved for the client app your partner will build.

## Backend

The backend is a Maven-based Spring Boot application. Run backend commands from the repository root so Maven uses the configured source paths.

## Frontend

The frontend folder is currently a placeholder for the UI project.

## Notes

- The existing backend code lives under `Backend/src` to keep the repo layout clean.
- Build output still goes to the root `target/` directory unless you change the Maven configuration later.
